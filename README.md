# 🐍🧩 נחש חידות

משחק Snake עם טוויסט - במקום לאכול תפוחים, פותרים חידות!

## איך משחקים

1. מופיעה חידה (מתמטיקה, לוגיקה, ידע כללי)
2. על המסך יש כמה מספרים - אחד נכון, השאר מלכודות
3. כוון את הנחש לתשובה הנכונה!

## שליטה

- **מחשב:** חצים או WASD
- **טלפון:** לחץ על המסך לכיוון שאתה רוצה (ביחס לראש הנחש)

## ניקוד

- ✅ תשובה נכונה: +10 נקודות × רמה
- ❌ תשובה שגויה: -5 נקודות, הנחש מתקצר
- 🎉 כל 5 חידות: עלייה ברמה (מהירות עולה)
